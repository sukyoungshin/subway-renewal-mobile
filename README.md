# Subway-renewal-mobile

프토폴리오 용도로 작업한 써브웨이 웹앱 리뉴얼 프로젝트입니다.<br/>

### Summary

- 인원 : 개인프로젝트 (1인)
- 기여 : 기획, 디자인, 프론트앤드
- 작업기간 : 2021.11. ~ 2022.02 (1차 작업 완료, 추가기능 구현중. (🔗 [프로젝트 소개](https://www.notion.so/f87299ddf4fa471a9def39386c7492ea) 참고)
